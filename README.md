## Project Workflow / Steps

1. **Data Loading**
   - Loaded dataset into Python using Pandas
   - Imported data into SQL databases (PostgreSQL/MySQL)

2. **Data Cleaning**
   - Handled missing values
   - Standardized column names
   - Corrected data types
   - Removed inconsistencies

3. **Exploratory Data Analysis (EDA)**
   - Analyzed customer demographics
   - Identified spending patterns
   - Studied discount and subscription impact
   - Performed category and age-based analysis

4. **SQL Analysis**
   - Wrote complex SQL queries using:
     - Joins
     - Subqueries
     - CTEs
     - Window functions
   - Segmented customers and calculated revenue metrics

5. **Dashboard Development**
   - Built an interactive Power BI dashboard
   - Visualized KPIs such as:
     - Revenue by age group
     - Category-wise sales
     - Subscription impact
     - Customer segmentation

6. **Reporting & Presentation**
   - Created a detailed analytical report
   - Designed a professional presentation (PPT) using Gamma

## Key Results & Insights
- Certain age groups contribute significantly higher revenue
- Subscribed customers tend to be repeat buyers
- Discount strategies influence purchasing behavior
- A small set of products dominates category-level sales
- Customer segmentation helps identify loyal and high-value customers
